# 🧠 Sudoku AI Solver

An AI-powered Sudoku Solver that efficiently solves puzzles using the Backtracking algorithm with constraint optimization, combined with a modern and interactive user interface.

---

## 🚀 Features

* 🔍 AI-based Sudoku solving using Backtracking
* 🎯 Efficient constraint checking for valid placements
* 🎨 Clean, modern, and aesthetic UI
* 🟩 Visual Feedback System:

  * Green → Correct prediction on first attempt
  * Red → Indicates backtracking intensity
* ⚡ Step-by-step solving visualization
* 🎮 Multiple difficulty levels (Easy / Medium / Hard)
* 🔄 New Game and Reset functionality

---

## 🛠️ Tech Stack

* **Frontend:** HTML, CSS, JavaScript

## ⚙️ How It Works

The solver uses a recursive Backtracking approach:

1. Finds empty cells in the Sudoku grid
2. Tries numbers from 1 to 9 based on constraints
3. Proceeds recursively if valid
4. Backtracks when conflicts occur
5. Continues until the puzzle is solved

The system also visually indicates how efficiently the AI reaches the solution using color-based feedback.
